This project creates a pressure sensor to monitor atmospheric pressure.
Hardware is a ESP-WROOM-32. Connected is a Bosch BMP280 and a IST AG Hygrometer. Connection is via I2C.
The data is processed and then uploaded to a volkszähler. IP of the volkszähler is 192.168.178.3
